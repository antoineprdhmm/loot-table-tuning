## Setup

Create a venv
```sh
python3 -m venv venv
```

Activate the venv
```sh
source venv/bin/activate
```

Install the dependencies
```sh
pip3 install -r requirements.txt
```

Run
```sh
python3 src/main.py
```

## Configuration

Feel free to change the values in `/config/dev.yaml`
